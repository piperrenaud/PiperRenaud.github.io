Wiremu's Tour

An educational visual novel game about New Zealand's wildlife, designed for students aged 10-12 and visitors learning about native animals.

Unity C#

Overview

Wiremu's Tour is an educational visual novel game where players explore different New Zealand animals through interactive dialogue and simple navigation.

The goal was to create an experience that felt approachable, colourful, and easy to understand for younger players, while also being suitable for tourists and classroom environments.

Project Info

Role

Programmer

Team

My Contributions

  • Programmed a custom dialogue system.
  • Built the animal selection system.
  • Created scene navigation between menu, animal selection, and story scenes.
  • Implemented game flow for selecting and completing animal stories
  • Helped make the experience accessible for the target audience.

Technical Challenges

Non-programmer editable dialogue

The project needed a dialogue system that writers could edit without needing to understand or touch the code.

I create a custom dialogue system that could be written and structured in a node-based format, making it easier for the narrative team to iterate.

Animal-based progression

Each animal needed its own education content and completion state.

I created a gameplay flow that tracks the selected animal and allows each story scene to load the correct content.

Key Features

Dialogue System

Interactive conversations using custom dialogue system.

Animal Selection

Players can select from a variety of New Zealand animals to explore.

Educational Content

Each animal has its own unique learning material and dialogue.

Simple Navigation

Designed to be easy for younger players to understand.

What I Learned

This project helped me understand how to build systems that support a team, not just the player. I learned how important it is to make tools and workflows that allow artists, writers, and designers to add content without relying on constant programmer support.
